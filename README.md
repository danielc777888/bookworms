# bookworms
Backend with Haskell, Servant, Postgresql and Datastar.

## install
- Install ghcup: _https://www.haskell.org/ghcup_
- Execute: _./setghc_

## run server
- Execute: _cabal run_

## formatters
- fourmolu: _https://github.com/fourmolu/fourmolu_
- cabal-gild: _https://github.com/tfausak/cabal-gild_
